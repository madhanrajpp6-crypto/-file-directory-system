# 📁 File Directory System (Mini Project)

## 📌 Description

This project simulates a File Directory System using a Tree Data Structure.
It allows users to create folders and files and display them in a hierarchical structure similar to a real file system.

---

## 🚀 Features

* 📁 Create Folder (mkdir)
* 📄 Create File (touch)
* 🌳 View Directory Structure (tree)
* 🎨 Attractive User Interface
* ⚡ Real-time updates

---

## 🧠 Concept Used

* Tree Data Structure
* Node representation (File / Folder)
* Parent-child relationship
* Recursive traversal

---

## 🛠️ Technologies Used

* Backend: Node.js, Express
* Frontend: HTML, CSS, JavaScript
* Data Storage: In-memory (Tree structure)

## 💡 How It Works

* Root directory is the main node.
* Each folder can contain child nodes.
* Files are leaf nodes.
* Data is stored in memory.
* Tree is displayed using recursion.
